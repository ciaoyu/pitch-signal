# 角球预测轻量模型

## 目标

基于球队历史数据 + 实时比赛进程 + 盘口线，给出角球 over/under 倾向分析。

---

## 一、计算流程

```
输入:
├── 球队历史场均角球数（主/客分开）
├── 对手场均被角球数
├── 双方风格加成/减值
├── 实时比赛角球节奏
├── 场馆/天气调整
└── 盘口线对比

输出:
├── 预测总角球数
├── over/under 倾向（+ 置信度）
├── 当前节奏 vs 预期节奏
└── 一句话分析
```

## 二、预测公式

### 2.1 基础预测

```
预测总角球 = 主队角球预期 + 客队角球预期

其中:
主队角球预期 = 主队场均角球(主场) × 对方场均被角球 / 联赛均值 × 风格系数
客队角球预期 = 客队场均角球(客场) × 对方场均被角球 / 联赛均值 × 风格系数
```

### 2.2 风格系数

| 球队风格 | 角球系数 | 说明 |
|---------|---------|------|
| 高位逼抢+边路进攻 | 1.25 | 边路传中多，角球多 |
| 传控中路渗透 | 0.85 | 短传渗透，角球偏少 |
| 防守反击 | 0.75 | 控球少，角球少 |
| 长传冲吊 | 1.35 | 频繁传中，角球多 |
| 均衡型 | 1.00 | 接近均值 |

### 2.3 实时调整

比赛开始后，用实时数据进行修正：

```
实时预测 = 基础预测 × 0.4 + (当前角球数 / 已进行分钟 × 90) × 0.6
```

权重动态调整：
- 比赛 0-15 分钟：基础预测权重 0.8，实时权重 0.2
- 比赛 15-30 分钟：基础预测权重 0.6，实时权重 0.4
- 比赛 30-60 分钟：基础预测权重 0.4，实时权重 0.6
- 比赛 60+ 分钟：基础预测权重 0.2，实时权重 0.8

### 2.4 场馆/天气修正

| 条件 | 修正值 |
|------|--------|
| 人工草皮（球速快） | +0.5 角球 |
| 暴雨/大风 | -1.0 角球 |
| 高温 30°C+ | -0.5 角球（节奏慢） |
| 高海拔 2000m+ | -0.5 角球（体能消耗大） |

## 三、Over/Under 判断逻辑

```
盘口线 = X.X（The Odds API 获取，fallback 用市场均值）
预测值 = Y.Y（模型计算）

倾向 = if |预测值 - 盘口线| < 0.5 → "neutral"
       if 预测值 - 盘口线 > 1.5 → "over_strong"
       if 预测值 - 盘口线 > 0.5 → "over_slight"
       if 盘口线 - 预测值 > 1.5 → "under_strong"
       if 盘口线 - 预测值 > 0.5 → "under_slight"

置信度 = if |预测值 - 盘口线| > 2.0 → "high"
         if |预测值 - 盘口线| > 1.0 → "medium"
         else → "low"
```

## 四、实时角球追踪条（前端组件数据）

```
{
  "matchId": "760416",
  "minute": 15,
  "currentCorners": { "home": 1, "away": 0, "total": 1 },
  "predictedTotal": 7.2,
  "oddsLine": 9.5,
  "expectedByNow": 1.2,      // 按预测到现在应该有 1.2 个
  "pace": "below",             // 当前节奏：below / on_track / above
  "trend": "under_favored",    // under_strong / under_slight / neutral / over_slight / over_strong
  "confidence": "medium",
  "paceDetails": {
    "cornersPer90": 6.0,       // 折合全场
    "progress": "10.5%",       // 盘口完成度
    "projection": "约 6-7 个全场角球"
  }
}
```

**角球进度条逻辑：**
```
进度% = 当前角球数 / 盘口线 × 100
颜色 = 进度 < 预期进度 → 🔵 under方向
       进度 ≈ 预期进度 → ⚪ 接近
       进度 > 预期进度 → 🔴 over方向

预期进度 = (当前分钟 / 90) × 100
```

## 六、数据来源与 Fallback

| 数据点 | 主来源 | Fallback |
|--------|--------|----------|
| 历史场均角球 | ESPN team stats | 联赛均值 9.5/场 |
| 球队风格 | 教练分析模块 | 默认均衡 1.0 |
| 盘口线 | The Odds API | 市场均值 9.5 |
| 实时角球 | ESPN match stats | 0（刚开赛） |
| 天气 | 天气 API | 默认无修正 |

---

## 七、准确率评估规划

| 累积比赛 | 评估指标 | 目标 |
|---------|---------|------|
| 10 场 | over/under 方向正确率 | > 55% |
| 20 场 | 盘口线误差 MAE | < 2.5 角球 |
| 48 场（小组赛完） | 方向正确率 + 误差 | > 60%, MAE < 2.0 |
