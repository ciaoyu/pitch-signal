# Owner H — 教练研究：数据字典与来源

> 基线 `78da1b5`，worktree `codex/coach-effect-research`。F/G 只读引用，外部池只读输入。

## 1. 研究级 tenure 数据要求（最小 schema）

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| `coach_id` | string | ✅ | 稳定 ID（跨国家队唯一） |
| `team_id` | string | ✅ | 国家队 ID（与 Elo / 赛果表对齐） |
| `start_date` | ISO date | ✅ | 任职开始（精确到日，至少年月） |
| `end_date` | ISO date \| null | ✅ | 离任日；null = 现任 |
| `source` | string | ✅ | 可追溯来源（URL / dataset / 文献） |
| `match_links` | array | ✅ | 该任期内每场比赛的 ID（用于残差） |
| `competitions` | array | ✅ | 预选赛 / 洲际杯 / 友谊赛 / 世界杯 覆盖标记 |
| `tactics_obs` | object | ⬜ | 阵型/压迫/推进/换人（可观测，非形容词） |

## 2. 仓库内现有数据

### 2.1 `data/coaches.json`（**展示级，非研究级**）

现状（抽样核对，见覆盖审计脚本真实产出）：

| 字段 | 现状 | 是否研究级 |
|------|------|-----------|
| `name` | 中文名 | ❌（需与权威 ID 对齐） |
| `nationality` | 中文 | ⬜ 辅助 |
| `age` | 数字 | ⬜ 辅助 |
| `style` | 中文关键词（"控球+高位压迫"） | ❌ **禁用**作特征 |
| `styleDetail` | 中文描述 | ❌ 禁用 |
| `formation` | 阵型数组（如 4-3-3） | ⬜ 可观测，但需逐场验证 |
| `since` | **仅年份**（"2022"） | ❌ 无精确起止日 |
| `tenure` | 中文（"4年"） | ❌ 无 end_date |
| `winRate` | 中文（"70%"） | ❌ 描述性，非残差 |
| `bigTournament` / `adjustment` / `vsOpponent` / `flexibility` / `notes` | 中文形容词 | ❌ **全部禁用**作特征 |

**结论**：`data/coaches.json` 是旧中文关键词模型的展示载体，**不满足研究级 schema 任何一条**（无起止日、无来源、无任期链、无场次级链接、无 xG）。H **不恢复**此模型，仅将其列为"展示现状"参考。

### 2.2 仓库内可复用对齐键

- `data/team_meta.json`：国家队元数据（用于 `team_id` 对齐）。
- `data/history/worldcup_*.json`：1930–2022 共 22 届赛果（**无教练字段**）。
- `data/venues.json`：2026 场馆（海拔/草坪/时区，供场地协变量）。
- Elo：仓库既有 Elo 序列（供实力基准）。

## 3. 外部只读输入（不提交）

| 源 | 内容 | 许可 | 接入方式 |
|----|------|------|---------|
| 国际赛果池（martj42/international_results 等 49k 级） | 国家队逐场赛果（含预选赛/洲际杯/友谊赛/世界杯） | 仓库各自许可，按需核对 | `COACH_RESEARCH_POOL_DIR` 指向，只读 |
| 教练任期历史（transfermarkt / wikipedia / 各足协档案） | 起止日期 + 来源 | 各自 CC/合理使用，需标注 | `COACH_RESEARCH_TENURE_DIR` 指向，只读 |
| xG / 阵容（F 产出，只读） | shot-xG、首发、出场分钟 | F 的 artifact | 仅引用，不修改 |

> 外部数据**绝不提交**进本分支；仅在 `ENV` 指向时由脚本读取，缺失时脚本显式 BLOCKED（不伪造）。

## 4. 缺失变量与缺失机制

| 缺失变量 | 缺失机制 | 处理 |
|----------|----------|------|
| 历史任期起止日（pre-2000 国家队） | **结构缺失**（档案不全） | 标记 `missing_reason='archive_gap'`；缺失 ≠ 中性，退化为"无教练效应"基准 |
| 早期比赛 xG | 无射门级数据 | 用 xg_diff 代理（净胜球/预期净胜球）并标 `proxy=true` |
| 战术可观测量（压迫/推进） | 需事件数据 | F/G 就绪前留空，不估算 |
| 弱队/小国任期 | 档案稀疏 | 分层收缩保护，后验区间宽 |

缺失机制**只有在 OOS 证明有信息**时才可作特征（治理原则）；否则作为"无效应"基准。

## 5. 许可与伦理

- 内部展示数据（CC0 / 项目源）可提交；外部源仅只读、标注出处。
- 教练增值为聚合统计，不涉及个人敏感数据；不发布未经验证的个体"神帅/水帅"标签。
