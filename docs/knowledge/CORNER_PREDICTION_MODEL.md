# 角球预测（实验性模型）

## 定位与边界

该模块提供可解释的角球估算和实时节奏对比。它尚未完成样本外校准，因此必须标记为“实验性估算”，不得包装成确定性预测或投注建议。

“参考线”和“盘口线”严格区分：

- `kind=reference`：人工录入的比赛比较基准，不是博彩公司盘口。
- `kind=market`：必须同时具备真实来源、角球线和 Over/Under 赔率才能使用。
- 没有可信线时返回 `line=null`，页面显示“暂无”，不得用固定数字补位。

法国—西班牙（ESPN `760514`）当前人工参考线为 `8.5`，记录于 `lib/corner-model.js`。

## 数据

- ESPN 本届世界杯已完赛 boxscore：`wonCorners`。
- 同一场 boxscore 中对手的 `wonCorners`：派生为 `cornersAgainst`。
- 本届赛事基线：截至 2026-07-15 已完赛 100 场，共 920 个角球，场均 `9.2`。这是模型先验，不是盘口线。
- 实时角球与比赛分钟：ESPN summary。

教练风格、草坪类型、天气描述不进入角球公式；当前没有验证证据支持这些手工系数。

## 赛前公式

```text
主队原始期望 = mean(主队场均赢得角球, 客队场均被角球)
客队原始期望 = mean(客队场均赢得角球, 主队场均被角球)

样本权重 = min(0.75, 双方最小有效样本数 / 8)
球队期望 = 赛事单队先验 + 样本权重 × (原始期望 - 赛事单队先验)
预测总角球 = 主队期望 + 客队期望
```

赛事单队先验为 `9.2 / 2 = 4.6`。每队结果限制在 `1.5–8.0`，用于防止小样本异常值直接产生极端输出。即使样本充足，仍至少保留 25% 的赛事均值收缩。

法国—西班牙当前输入示例：

```text
法国：场均赢得 6.8，场均被 3.0（6 场）
西班牙：场均赢得 7.3，场均被 1.2（6 场）

法国原始期望 = (6.8 + 1.2) / 2 = 4.0
西班牙原始期望 = (7.3 + 3.0) / 2 = 5.15
样本权重 = 0.75
收缩后预测 ≈ 4.2 + 5.0 = 9.2
```

旧公式会把双方进攻角球均值互相相乘，错误输出 `20.9`；该路径已删除。

## 实时调整

```text
0–15 分钟：赛前估算 80% + 当前节奏投影 20%
16–30 分钟：赛前估算 60% + 当前节奏投影 40%
31–60 分钟：赛前估算 40% + 当前节奏投影 60%
61 分钟后：赛前估算 20% + 当前节奏投影 80%
```

早期角球节奏波动很大，因此即使和参考线偏差较大，当前实验模型置信度最高只到 `medium`，不会输出 `high`。

## API 契约

`GET /api/corner-analysis/:matchId`

关键字段：

```json
{
  "historical": {
    "homeAvg": 6.8,
    "awayAvg": 7.3,
    "homeAgainstAvg": 3.0,
    "awayAgainstAvg": 1.2,
    "homeSampleCount": 6,
    "awaySampleCount": 6
  },
  "predicted": {
    "total": 9.2,
    "status": "experimental_unvalidated",
    "method": "corners_for_plus_opponent_conceded_shrunk_to_tournament_mean"
  },
  "odds": {
    "line": 8.5,
    "kind": "reference",
    "source": "manual_reference",
    "over": null,
    "under": null
  }
}
```

## 后续验证

- 永久保存每场赛前输入、输出、模型版本和最终角球。
- 报告 MAE、参考线方向命中率和分层样本量。
- 只有通过时间切分的样本外验证后，才允许升级置信度或加入新特征。
